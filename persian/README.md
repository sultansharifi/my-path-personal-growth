# مسیر من — دفتر شخصی رشد و خودآگاهی

یک وب‌اپ خصوصی برای ثبت اشتباهات شخصی، شناخت محرک‌ها، مشاهده الگوهای تکرار و تبدیل هر تجربه به یک تعهد کوچک و عملی. این محصول سازمانی نیست و هیچ مفهوم کارمند، مدیر یا مجازات اداری ندارد.

## امکانات

- داشبورد شخصی «امروز من»
- تعریف و مشاهده الگوهای رفتاری
- ثبت اتفاق همراه با احساس، محرک و یادگیری
- ساخت خودکار تعهد ترمیمی پس از هر ثبت
- دفترچه رشد و ثبت بردهای کوچک
- نمودار روند هفتگی و بینش‌های شخصی
- تاریخچه کامل و قابل جست‌وجو
- یادآور مرور روزانه، Dark Mode و رابط کامل RTL
- ماندگاری اطلاعات نسخه نمایشی پس از Refresh
- مدل PostgreSQL/Prisma مخصوص اطلاعات یک کاربر

## اجرا

```bash
npm install
copy .env.example .env
npm run db:generate
npm run db:migrate
npm run db:seed
npm run dev
```

برای اجرای migration، آدرس PostgreSQL را در `DATABASE_URL` تنظیم کنید. رمز حساب نمونه هنگام seed با bcrypt هش می‌شود.
